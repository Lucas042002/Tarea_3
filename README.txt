# Tarea_3

CONSIDERACIONES DE EJECUCIÓN

A considerar: el archivo proyecto "Tarea 3" fue creado en Visual Studio Code.

Para que funcionen las distintas opciones del menú, debe ingresar primero la opcion 1: Importar Archivo, acto seguido debera ingresar
la cantidad de lineas que texto que desea leer, sino no podrá acceder a ciertas funciones que requieran del archivo en si.

Para valores no númericos el programa presenta bucles infinitos en el menú, dado que no corresponde el tipo de opción (está descrito como variable numérica).

Para la búsqueda, deben ser ingresados los parámetros de búsqueda exactamente como es requerido al agregar una nueva ruta. ej: Si el nombre de la ruta es RuTaaa2 usted debe ingresar
RuTaaa2 no rutaaa2, ya que no esta estandarizado.
